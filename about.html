This website summarizes information about hypertext systems. The information is mainly queried and visualized from Wikidata.

Contributions

Contributions and comments are welcome! Please directly edit Wikidata to add missing systems, people, connections, and pictures! You can also leave a comment in this website’s issue tracker.

Related projects

The Historic Hypertext Project (@hist_HT), coordinated by Claus Atzenbeck, aims at collecting and running historic hypertext systems in virtual machines.

Wikidata for Digital Preservation (@WikiDigi) curates information about about the domain of computing (file formats and software, …) in Wikidata.

An instance of hypertext

Unlike most hypertexts this website makes explicit use of transclusion, the core feature of Ted Nelson's original hypertext design:

  • Diagrams are transcluded with an HTML iframe from Wikidata Query Service. Each SPARQL query is kind of an edit decision lists.
  • HTML section (such as the one you are reading) are transcluded via JavaScript from individual files.

Follow the “sources” links in the right upper corner to the current SPARQL/HTML document. The git repository further contains versioning information. This is far from perfect but more xanalogical then most other hypertexts.
