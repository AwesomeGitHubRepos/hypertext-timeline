This website uses several SPARQL-Queries against Wikidata to visualize information about hypertext systems. Please directly edit Wikidata to add missing systems, people, connections, and pictures!

Sources

The source code of this website is hosted in a GitHub project including an issue tracker. Contributions and comments are welcome!

This website is also an instance of hypertext. Unlike most hypertexts it also makes explicit use of transclusion, the core feature of Ted Nelson's original hypertext design:

  • Diagrams are generated from data queried via SPARQL queries from Wikidata and transcluded via an HTML iframe from the Wikidata Query Service. One could describe the queries as kinds of edit decision lists.
  • HTML section (such as the one you are reading) are transcluded via JavaScript from individual files.
Use the SPARQL and/or HTML link at the right upper corner to get to the sources.
